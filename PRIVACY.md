# Privacy Policy

**Last updated**: 2026-05-08

## Overview

`claude-project-bootstrap` is an open-source Claude Code plugin that runs entirely on your local machine. It does not collect, transmit, or store any personal data.

## Data Collection

This plugin collects **no data**. Specifically:

- **No analytics or telemetry** — No usage data is sent anywhere.
- **No network requests** — The plugin operates entirely offline, reading and writing files only in your local project directory.
- **No accounts required** — No sign-up, login, or authentication is needed.
- **No cookies or tracking** — The plugin has no web component.

## What the Plugin Does

The plugin generates project configuration files (CLAUDE.md, pre-commit hooks, test framework docs, etc.) based on your interactive input during `/init-project`. All generated files remain in your local project directory.

## Third-Party Services

This plugin does not integrate with or send data to any third-party services. It runs within the Claude Code environment, which is governed by [Anthropic's Privacy Policy](https://www.anthropic.com/privacy).

## Open Source

The full source code is available at [github.com/seungjaeyuu/claude-project-bootstrap](https://github.com/seungjaeyuu/claude-project-bootstrap) under the MIT license. You can inspect exactly what the plugin does.
